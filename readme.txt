This plugin adds a custom post type labeled Events, and integrates with your google calendar.
== Description ==

Available translations:

* English

== Installation ==

1. Upload `google-calendar-int` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure and enjoy :)
